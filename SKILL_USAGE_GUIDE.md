# 🚀 Agent Skills 使用指南 (User Guide)

恭喜！您的專案已經安裝了 **UIUX Pro Max** 與 **Superpowers** 兩大強力套件。這份指南將協助您有效運用這些能力。

---

## 🎨 1. UIUX Pro Max (極致設計)

**核心功能**：這是一個「設計大腦」。當您需要建立介面時，它不只是寫程式碼，而是會先產生一套完整的**設計系統** (配色、字體、間距、互動規範)。

### 如何使用
您不需要記住複雜的指令，只需在對話中明確表達您的設計需求：

*   **建立新頁面**：
    > "幫我設計一個 SaaS 產品的 Landing Page，風格要像 Linear 那樣簡約現代。"
    > (Agent 會自動啟動設計系統生成器，選擇適合的字體與配色)

*   **指定風格**：
    > "這個 Dashboard 請使用 Glassmorphism (毛玻璃) 風格。"
    > "我想做一個 Cyberpunk 風格的個人網站。"

*   **優化現有介面**：
    > "請用 UIUX Pro Max 的標準幫我優化目前的登入表單，讓它更有質感。"

### 它會做什麼？
1.  **生成設計規範**：自動決定主色、輔助色、字體大寫/行高。
2.  **微互動建議**：告訴您按鈕 Hover 時應該上浮幾像素、陰影如何變化。
3.  **防雷機制**：避免使用不適合該產業的設計 (例如：在銀行 App 使用霓虹燈配色)。

## ⚡ 2. Superpowers (超能力工程流)

**核心功能**：這是一套嚴謹的**軟體工程工作流**。它改變了 Agent 寫程式的方式——從「直接寫 Code」轉變為「思考 ➡️ 規劃 ➡️ 執行 ➡️ 驗證」。

### 自動化工作流
這些技能通常會根據您的請求自動觸發，但您也可以明確要求：

#### Phase 1: 需求分析 (Brainstorming)
當您有一個模糊的想法時：
> **指令**："我想做一個[功能]，請先幫我 Brainstorming。"
> **效果**：Agent 不會馬上寫程式，而是會用蘇格拉底式提問法，釐清您的需求、邊界情況 (Edge Case) 和潛在問題。

#### Phase 2: 規劃 (Writing Plans)
當需求確認後：
> **指令**："請幫我寫一份詳細的實作計畫 (Implementation Plan)。"
> **效果**：Agent 會產出一份步驟清單，將大任務拆解為 2-5 分鐘可完成的小任務。

#### Phase 3: 執行與測試 (TDD & Execution)
在寫程式階段：
> **指令**："請依照計畫執行，並嚴格遵守 TDD (測試驅動開發)。"
> **效果**：Agent 會先寫測試 (紅燈)，再寫程式讓測試通過 (綠燈)，最後重構。

#### Debugging (系統化除錯)
當遇到 Bug 時：
> **指令**："請用系統化除錯 (Systematic Debugging) 流程來修這個 Bug。"
> **效果**：Agent 會提出假設 ➡️ 驗證假設 ➡️ 修正 ➡️ 確認修復，而不是盲目嘗試。

## 🌟 最佳實踐：雙劍合璧

最強大的用法是將兩者結合。試試看這樣的 Prompt：

> "我想開發一個**全新的專案管理 Dashboard**。
> 1. 請先用 **Superpowers: Brainstorming** 幫我釐清核心功能與使用者流程。
> 2. 確認需求後，請用 **UIUX Pro Max** 生成一套現代化的設計系統。
> 3. 最後，請幫我寫一份實作計畫並開始開發。"

這將確保您的產品不僅**結構穩健** (Superpowers)，而且**外觀頂級** (UIUX Pro Max)。
